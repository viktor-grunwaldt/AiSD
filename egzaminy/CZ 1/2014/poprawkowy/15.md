### Treść
(DP)
Zapisz w pseudokodzie algorytm znajdujący długość najdłuższego wspólnego
podciągu dwóch ciągów.

------
<details><summary>Rozwiązanie</summary>
  
```python
# postać ciągóœ
S1 = x1...xn
S2 = y1...ym

# Zerujemy pierwszy wiersz i kolumnę dp
dp[0][0...m] = 0
dp[0...n][0] = 0

def lcs(n, m):
  for i in range(1, n):
    for j in range(1, m):
      if S1[i] == S2[j]:
        dp[i][j] = 1 + dp[n-1][m-1]
      else:
        dp[i][j] = max(dp[i-1][j], dp[i][j-1])
  return dp[n, m]
  
```
